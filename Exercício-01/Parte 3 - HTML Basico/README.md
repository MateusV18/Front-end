# Exercício 1 - Parte 3: HTML Básico

Este repositório contém a resolução da Parte 3 do Exercício 1, focada na estruturação de uma página web utilizando elementos básicos do HTML.

## Explicação das Tags Utilizadas

Abaixo está a descrição do papel de cada tag HTML implementada neste projeto:

* `<!DOCTYPE html>`: Informa ao navegador que o documento está sendo escrito na versão mais recente do HTML (HTML5)
* `<html>`: A tag mais importante onde contêm todo o conteúdo HTML formado pelo `<body>` e o `<head>` `lang="pt-BR"` defini o idioma da página para português Brasil.
* `<head>`: Contém os metadados e configurações da página (informações das páginas que o navegador acessa mas não exibi diretamente no corpo da página como: título da página, codificação de caracteres, descrição da página,palavras chave ,links externos de estilização).
* `<meta charset="UTF-8">`: Define a codificação de caracteres para garantir que sejam exibidos corretamente.
* `<meta name="viewport" ...>`: Configura a janela de visualização da página, para que se adapte ao tamanho real da telas do dispositivo como celular e tablet, tornando a página responsiva.
* `<title>`: Define o título da página que aparece na aba do navegador.
* `<body>`: Contém todo o conteúdo visível do site (textos, imagens, listas, etc.).
* `<h1>`: Define o título principal da página. Deve ser usado apenas uma vez por página para manter uma boa hierarquia e SEO.
* `<h2>`: Defini os subtítulos da página, ajuda a criar seções na páginas com títulos menos importante que o título principal `<h1>`.
* `<p>`: Representa um parágrafo de texto comum.
* `<ul>`: Cria uma lista não ordenada.
* `<ol>`: Cria uma lista ordenada.
* `<li>`: Define um item de linha dentro de uma lista.
